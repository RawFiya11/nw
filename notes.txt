make a new branch

1. git checkout -b branch_name

create a new file

1. touch file_name

open file in subl

1. subl file_name

save and close

1. save - command+s
2. close - command+w

commit changes to repository

1. git status
2. git add .
3. git commit -m "message here"
4. git push origin branch_name

submit a pull request

1. go to github webpage
2. in repository, click compare and pull request 
3. add coments if needed, click create pull request
4. review code via commits tab
5. add baxline to assignes 
6. if everything looks good, click merge pull request 
7. click confirm
8. click delete branch if wanted

delete a branch on local computer

1. git checkout master
2. git branch -D branch_name
